# 🎨 Traderra — Design System (Dark Studio Theme)

## 1. Overview

The **Traderra Design System** defines the visual and interactive identity of the platform. It translates the brand’s professional and truthful philosophy into an elegant, immersive, and functional visual language.

### Core Principles
1. **Professional Elegance:** Minimalist, institutional-grade aesthetics.  
2. **Cinematic Immersion:** Visual depth through subtle gradients and glow.  
3. **Functional Precision:** No excess — every component serves usability.  
4. **Visual Truth:** Design avoids emotional bias (e.g., no exaggerated red/green signals).  

The aesthetic foundation is **Dark Studio**, built on grayscale layers with **gold accents** symbolizing precision, mastery, and value.

## 2. Visual Identity

### 2.1 Color Palette

| Color Role | Name | Hex | Description |
|-------------|------|-----|-------------|
| **Primary Background** | Deep Graphite | `#0E0E10` | Base for all screens; pure dark tone |
| **Surface Layer 1** | Iron Slate | `#1A1A1C` | Card backgrounds, panels, modals |
| **Surface Layer 2** | Soft Charcoal | `#2C2C2F` | Hover and secondary layers |
| **Text Primary** | Off-White | `#EDEDED` | Soft white for high readability |
| **Text Secondary** | Mist Gray | `#A6A6A6` | Supporting text, placeholders |
| **Accent** | Gold | `#CBA135` | Used sparingly for active states, highlights |
| **Success** | Muted Green | `#4BAA7A` | Non-aggressive success tone |
| **Warning** | Amber Gold | `#E0A245` | Alert tone, matches gold family |
| **Error** | Warm Red | `#C05C4D` | Non-harsh red for clarity |

### 2.2 Color Philosophy
- All colors are slightly desaturated for reduced eye strain.  
- Brightness contrast is balanced to maintain night-time usability.  
- Gold accents are used in less than 10% of UI surface to maintain visual sophistication.

## 3. Typography

| Use Case | Typeface | Weight | Size Range |
|-----------|-----------|---------|-------------|
| Headers | **Inter / Satoshi** | 500–700 | 18–36px |
| Body | **Inter / Satoshi** | 400–500 | 14–18px |
| Labels / Buttons | **Inter** | 600 | 12–14px |
| Data Values | **Inter Mono** | 500 | 13–15px |

**Guidelines:**
- All typography uses slightly increased letter spacing for readability on dark backgrounds.
- Numeric data uses monospaced variant for alignment across rows and charts.
- Line height for paragraphs: 1.5x.

---

## 4. Layout & Spacing

### 4.1 Grid System
- **12-column grid**, responsive to screen width.  
- Gutter width: 24px desktop / 16px tablet / 12px mobile.  
- Content max width: 1440px.  
- All cards and containers have consistent corner radius (16px / 2xl).  

### 4.2 Spacing Scale
| Token | Value | Use Case |
|--------|--------|----------|
| `xs` | 4px | Icon padding |
| `sm` | 8px | Text spacing |
| `md` | 16px | Standard element gap |
| `lg` | 24px | Section separation |
| `xl` | 32px | Layout margin |
| `2xl` | 48px | Page-level breathing space |

## 5. Components

### 5.1 Cards
- Primary structural element.  
- Rounded 16px, soft shadow (`0 2px 12px rgba(0,0,0,0.4)`).  
- Background: Iron Slate.  
- Hover: Soft Charcoal.

### 5.2 Buttons
| Variant | Background | Text | Border | Hover |
|----------|-------------|------|--------|--------|
| Primary | Gold | Black | None | Lighter gold tone |
| Secondary | Transparent | Gold | 1px solid gold | Gold + transparency |
| Ghost | Transparent | Mist Gray | None | Gold text on hover |

Rounded corners (12px). Padding: 12px 20px.

### 5.3 Toggles
Two main toggle groups:
- **Gross / Net**  
- **% / $ / R**  

Each toggle is a pill button with subtle glow around the active state.

### 5.4 Navigation
- Fixed left sidebar, collapsible.  
- Top bar: quick actions + toggles.  
- Icons (Lucide) minimal line style.

**Sidebar sections:**
1. Dashboard  
2. Trades  
3. Journal  
4. Reviews  
5. AI / Renata  
6. Settings

### 5.5 Charts
- Built with Recharts / D3.  
- Dark background, gold or muted green line accents.  
- Axis labels use off-white; grid lines are semi-transparent gray.  
- Tooltip follows mouse position with fading animation.

### 5.6 Tables
- Alternating row shades for legibility.  
- Hover highlight in gold tint.  
- Headers in Mist Gray; numeric data in Off-White.

---

## 6. Motion & Interaction

- **Framework:** Framer Motion.  
- **Speed:** 200–350ms easing for most interactions.  
- **Style:** Ease-out cubic transitions for calm, professional movement.

### Key Motion Types
| Animation | Description |
|------------|--------------|
| Fade-In | Page load and modal entrance |
| Slide-Up | Buttons and cards entering viewport |
| Pulse Glow | Subtle focus highlight for active toggles |
| Hover Lift | 3px elevation increase on hover |

All motion avoids excessive bounce or spring for professionalism.

---

## 7. Accessibility & Readability

- Minimum contrast ratio: 7:1 for text and critical data.  
- Gold accent used only on high-contrast surfaces.  
- All inputs and toggles accessible via keyboard navigation.  
- Focus rings: 1px solid gold.  
- Optional high-contrast mode disables gradients and motion.

## 9. Chart Design Language

### Core Chart Types
| Chart | Purpose | Notes |
|--------|----------|-------|
| Equity Curve | Track cumulative P&L | Smooth gold line, soft gradient fill |
| Daily P&L Bars | Show daily performance | Bars alternate gold/gray |
| Distribution Histograms | Analyze expectancy, R-multiple | Neutral gray bins, gold mean line |

### Chart Behaviors
- Hover to reveal data points + stat tooltips.  
- Double-click to zoom reset.  
- Gold glow pulse on highlight.

---

## 10. Tone Integration in Design

Renata’s tone and personality subtly influence UI cues:
- **Analyst Mode:** Straight lines, minimal motion, no color fluctuation.  
- **Coach Mode:** Slight motion ease, small gold accent increase.  
- **Mentor Mode:** Warmer gold hue, slightly smoother gradients.  

This creates a subconscious emotional alignment between data tone and interface tone.

## 13. Responsive Design Principles

| Breakpoint | Layout Adjustment |
|-------------|------------------|
| `>1440px` | Full desktop grid (12-column) |
| `1024–1440px` | Compact grid (10-column) |
| `768–1024px` | Collapsible sidebar, reduced chart density |
| `<768px` | Vertical stack layout, adaptive AI chat pane |

All elements scale fluidly with rem units. Charts dynamically resize with parent containers.
