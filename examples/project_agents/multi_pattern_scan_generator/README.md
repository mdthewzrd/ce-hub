# Multi-Pattern Scan Generator

Generates diverse trading scanners from natural language descriptions using proven pattern library (backside_b, lc_d2, a_plus_para, sc_dmr). Based on ScanCreatorAgent + Pattern Library integration.

## Installation

```bash
pip install -r requirements.txt
```

## Configuration

Set up environment variables:

```bash
# For Neo4j (if using)
NEO4J_URI=bolt://localhost:7687
NEO4J_USERNAME=neo4j
NEO4J_PASSWORD=password

# For OpenAI embeddings (if using)
OPENAI_API_KEY=sk-...

# For LLM (if using)
ANTHROPIC_API_KEY=sk-...
# or
OPENAI_API_KEY=sk-...
```

## Usage

```python
from multi-_pattern__scan__generator_agent import Multi-PatternScanGeneratorAgent

async def main():
    # Create agent
    agent = Multi-PatternScanGeneratorAgent()

    # Connect to vector database
    await agent.connect_vector_db()

    # Execute task
    result = await agent.execute(
        task="Your task here",
        use_knowledge=True  # Enable RAG
    )

    print(result)

    # Close connection
    await agent.close_vector_db()

if __name__ == "__main__":
    import asyncio
    asyncio.run(main())
```
