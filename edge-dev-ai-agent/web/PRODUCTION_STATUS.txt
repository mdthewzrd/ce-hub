================================================================================

To deploy to production:

1. Choose hosting platform:
   - Vercel (recommended): vercel --prod
   - Railway: railway up
   - AWS: Amplify or ECS

2. Set environment variables:
   - OPENROUTER_API_KEY
   - NEXT_PUBLIC_BACKEND_URL
   - NEXT_PUBLIC_EDGEDEV_URL

3. Configure domain:
   - Add custom domain
   - Configure DNS
   - Setup SSL

4. Enable monitoring:
   - Set up error tracking
   - Configure analytics
   - Set up alerts
