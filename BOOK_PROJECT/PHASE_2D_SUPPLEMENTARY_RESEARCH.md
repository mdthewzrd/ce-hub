### Copyright and Licensing Considerations

**Public Domain**:
- No attribution required
- Free to use and modify
- Verify PD status before using

**Creative Commons**:
- CC0: No attribution required
- CC BY: Attribution required
- CC BY-SA: Attribution + share-alike
- CC BY-ND: Attribution + no derivatives

**Commercial Licenses**:
- Royalty-free: One-time fee, perpetual use
- Rights-managed: Usage restrictions apply
- Exclusive: Higher cost, exclusive rights

**Fair Use Considerations**:
- Educational purpose favors fair use
- Transformative use (adaptation, commentary)
- Small portion of original work
- No market harm to copyright holder

**Attribution Best Practices**:
- Include in figure captions
- Add to bibliography/credits
- Maintain consistent format
- Verify all attributions before publication
