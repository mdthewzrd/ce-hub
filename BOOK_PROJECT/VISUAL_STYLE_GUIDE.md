## QUALITY STANDARDS

### Technical Accuracy
- All mathematical proportions must be precise
- Scientific diagrams must be verified
- Historical references must be accurate
- Measurements must be clear and correct

### Visual Clarity
- Primary elements must be immediately recognizable
- Labels must be readable at target size
- Colors must pass contrast ratios (WCAG AA minimum)
- Complexity must not overwhelm understanding

### Aesthetic Consistency
- All diagrams must share visual language
- Color usage must be purposeful and consistent
- Typography must follow hierarchy
- Effects must enhance, not distract

### File Specifications
- **Resolution:** 300 DPI minimum (print), 72 DPI (digital)
- **Format:** SVG for diagrams (scalable), PNG for raster
- **Color Space:** sRGB for digital, CMYK for print
- **File Size:** Optimized but quality prioritized
