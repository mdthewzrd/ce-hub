## ‚úèÔ∏è LINE STYLE SPECIFICATIONS

### Line Weights
```
ULTRA-THIN: 0.5-1px
  Usage: Fine details, subtle guides

THIN: 1-1.5px
  Usage: Labels, secondary elements, pattern overlays

MEDIUM: 2-3px (STANDARD)
  Usage: Main elements, primary shapes, core diagrams

THICK: 4-5px
  Usage: Emphasis, chapter dividers, key concepts
```

### Line Characteristics
- **Style:** Clean, confident, slightly rounded corners (radius: 2-3px)
- **No:** Sketchy/hand-drawn, ultra-thin technical, thick cartoon
- **Yes:** Precise but flowing, purposeful weight variation, intentional
- **End caps:** Round (for softer feel) or square (for technical diagrams)
- **Joins:** Round (smooth connections)

### Line Colors by Context
```
WHITE LINES: #ffffff
  On: Dark backgrounds, energy visualizations
  Weight: 1.5-3px depending on importance

DEEP BLUE LINES: #1a237e
  On: Light backgrounds, educational diagrams
  Weight: 2-3px for main elements

GOLD LINES: #ffd700
  Usage: Emphasis, key proportions, special highlights
  Weight: 2-4px

VIOLET LINES: #7b1fa2
  Usage: Spiritual concepts, chakras, transcendent ideas
  Weight: 1.5-2.5px

GRAY LINES: #9e9e9e
  Usage: Secondary elements, guides, construction lines
  Weight: 0.5-1.5px
```

## üìê ILLUSTRATION CATEGORY STYLES

### Category 1: Sacred Geometry Diagrams

**Visual Style:**
- Background: Dark Space (#0a0e27)
- Lines: White (1.5px) for main shapes, Gold (2.5px) for key proportions
- Glow: Subtle inner glow on major elements
- Precision: Perfect geometric accuracy
- Labels: Space Mono, small, gold

**Example - Flower of Life:**
- 19+ overlapping circles
- Each circle: 1.5px white line, 50% opacity
- Central flower: 2.5px gold line, 80% opacity
- Background: #0a0e27
- Subtle blue glow (radial gradient) behind entire pattern
- No fill (transparent)

---

### Category 2: Number Maps & Mathematical Patterns

**Visual Style:**
- Background: Light Parchment (#faf5e6)
- Lines: Deep Blue (#1a237e), 2-3px
- Numbers: Space Mono, evenly spaced, same size
- Flow arrows: Gradient fills, subtle
- Labels: Small, technical, Deep Blue

**Example - Vortex Math Number Map:**
- Circle: 2px Deep Blue line
- Numbers 1-9: Space Mono, 24pt, Deep Blue
- 1-2-4-8-7-5: Connected with blue gradient arrows (clockwise)
- 3-6-9: Connected with Gold (#ffd700), pyramid formation
- Small dots (3px) showing flow direction
- Background: #faf5e6 with subtle texture

---

### Category 3: Natural Pattern Overlays

**Visual Style:**
- Base: Original reference photo
- Overlay: Semi-transparent white/gold lines
- Opacity: 60-70% (photo still visible)
- Line weight: 1.5-2px
- Color: White or Gold depending on photo

**Process:**
1. Start with reference photo
2. Create clean tracing of pattern
3. Set opacity to 60-70%
4. Ensure photo still visible underneath
5. Add label if needed (small, Space Mono)

**Example - Nautilus Shell:**
- Photo: High-quality nautilus cross-section
- Overlay: Golden spiral tracing, white, 1.5px
- Opacity: 65%
- Optional: Phi proportion markers in gold
- Label: Small gold text "œÜ = 1.618..." in corner

---

### Category 4: Energy & Plasmoid Visualizations

**Visual Style:**
- Background: Dark Space (#0a0e27)
- Energy fields: Gradient fills (blue to violet)
- Core elements: Bright white/gold with glow
- Field lines: Semi-transparent white (30-50%)
- Glow effects: Inner and outer glow
- Motion: Subtle gradient arrows for flow

**Example - Plasmoid Structure:**
- Core: Bright white/gold center (#ffd700), radial glow
- Magnetic field: Violet gradient (#7b1fa2), radial, 40% opacity
- Field lines: Thin white (1px), showing containment
- Labels: Small Space Mono, white, positioned around
- Cross-section: Semi-transparent outer layer (30%)
- Energy flow: Subtle gradient arrows (blue ‚Üí violet)

---

### Category 5: Before/After Comparisons

**Visual Style:**
- Layout: Side-by-side or top-bottom split
- "Before": Original photo, full color, no overlays
- "After": Same photo with pattern overlay
- Divider: Clean gold line (2px)
- Labels: "What You See" / "What's Actually There"
- Arrow or gradient showing transformation

**Example - Sunflower Pattern Reveal:**
- Left: Original sunflower photo, no treatment
- Right: Same photo with golden spiral overlay
- Divider: Vertical gold line, 2px
- Label Left: "What You See" (Inter, white)
- Label Right: "Pattern Revealed" (Inter, gold)
- Optional: Gradient arrow (left to right)

### Category 6: Quote Graphics

**Visual Style:**
- Background: Full-bleed relevant image
- Image treatment: Darkened or blurred for readability
- Quote text: Cormorant Garamond, large, centered

- Overlay: Semi-transparent dark gradient (50-70%)
- Brand element: Small sacred geometry icon
- Size: Vertical (1080x1350) for Instagram

## üéØ QUALITY STANDARDS

### Technical Specifications
- **Format:** SVG preferred (scalable), PNG backup (300 DPI for print)
- **Colors:** Always use hex codes from this guide
- **Transparency:** Supports PNG/SVG with alpha channel
- **Resolution:** Minimum 300 DPI for print, 72 DPI for digital
- **File size:** Optimize for web (under 500KB per image where possible)

### Consistency Requirements
- [ ] Colors match brand palette exactly
- [ ] Line weights follow category specifications
- [ ] Typography uses defined font families
- [ ] Labels use appropriate font sizes
- [ ] Background colors match category style
- [ ] Glow effects are consistent (subtle, not overpowering)
- [ ] Geometry is precise (perfect circles, equal spacing)
- [ ] Text is legible (contrast ratio 4.5:1 minimum)

## üì± MOBILE OPTIMIZATION

Since Instagram is primary traffic source:

### Image Dimensions
```
FEED POSTS: 1080x1080px (square)
STORIES: 1080x1920px (vertical, 9:16)
CAROUSEL: 1080x1080px (each slide)
QUOTE GRAPHICS: 1080x1350px (4:5 vertical)
PRINT SPREAD: 2400x3600px (8x12", 300 DPI)
```

### Mobile Reading Considerations
- Text size: Minimum 16pt for body, 24pt for headers
- Contrast: High contrast for readability on small screens
- Simplicity: Avoid clutter, focus on single concept per image
- Touch targets: If interactive, minimum 44x44px

### Instagram Optimization
- Shareable quote graphics (designed to be saved and shared)
- Consistent visual style (recognizable brand)
- Vertical format preferred (better for mobile)
- Text overlay on images (concept + caption in one)
- Chapter opening spreads (dramatic, scroll-stopping)

## üîß PRODUCTION WORKFLOW

### 1. Concept Phase
- Sketch idea (can be rough)
- Define category and style
- Choose color palette
- Select typography

### 2. Reference Phase
- Find visual reference (scrape/collect)
- Save to /references/ folder
- Create spec document
- Log in master spreadsheet

### 3. Creation Phase
- Use AI agent to trace/recreate reference
- Apply brand style (colors, lines, fonts)
- Refine and iterate
- Save versions to /working/ folder

### 4. Quality Phase
- Check against quality standards
- Verify brand consistency
- Test on mobile if applicable
- Get approval if needed

### 5. Production Phase
- Export to final format (SVG + PNG)
- Optimize file size
- Move to /production/ folder
- Update master spreadsheet
- Archive working files
