## SEARCH STRATEGY TIPS

### For Golden Ratio/Phi Images:
1. Search "golden ratio" + specific subject
2. Try both "phi" and "golden ratio" (different results)
3. Include "fibonacci" for spiral patterns
4. Add "SVG" or "vector" for clean graphics

### For Nature Images:
1. Use scientific names (helianthus, nautilus pompilius)
2. Add "macro" or "close-up" for detail
3. Include specific fibonacci numbers (8, 13, 21, 34, 55)
4. Try both common and scientific names

### For Sacred Geometry:
1. Always search "SVG" first (vector = infinite quality)
2. Try "vector graphics" too
3. Search specific patterns (Seed of Life vs Flower of Life)
4. Look for "construction" diagrams (show how it's made)

### For Architecture/Art:
1. Include "golden ratio overlay" to see φ lines
2. Try "analysis" or "composition"
3. Search artist name + artwork + "golden ratio"
4. Look for educational diagrams

- Openclipart.org
- SVGsilh.com
- Publicdomainvectors.org

### Photos (Free):
- Unsplash.com (high quality photos)
- Pexels.com
- Pixabay.com (mixed, check quality)
- Wikimedia Commons (huge, check licenses)

### Premium (If Needed):
- Shutterstock.com
- Getty Images
- Adobe Stock
- iStock

### Educational/Academic:
- Wikimedia Commons (best for accuracy)
- University websites (search: site:.edu)
- Museum websites (search: site:.gov)
- NASA (for space images)
